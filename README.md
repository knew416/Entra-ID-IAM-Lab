Project Overview
This project demonstrates the implementation of Identity and Access Management controls using Microsoft Entra ID.

Business Scenario
Simulated a small organization with HR, IT, and Finance departments.

User Management
Created departmental users and implemented structured naming conventions.

Group-Based Access Control

Created Security Groups:

HR-Group
IT-Group
Finance-Group

Role-Based Access Control (RBAC)
Assigned built-in administrative roles using least privilege principles.
